# -CleanArchitecture-WorkerService
Worker Service criado com .NET 6, utilizando ainda SQL Server + Dapper.Contrib para gravação dos dados e um Dockerfile para geração de imagens Docker em Linux. Inclui configurações para monitoramento via Application Insights.
